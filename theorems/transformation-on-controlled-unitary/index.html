Transformation on controlled unitary

Theorem statement

If \(u\) is an eigenstate of \(U\) with eigenphase \(\theta\), then the YZZYZ quantum phase processor built from controlled \(U\) acts on \(\psi\otimes u\) as the corresponding single-qubit trigonometric QSP matrix at \(\theta\), up to the global phase \((e^{i\theta/2})^L\), tensored with \(u\).
