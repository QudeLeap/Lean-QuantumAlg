Bernstein-Vazirani algorithm

Theorem statement

For every hidden string \(s\), running the Walsh-Hadamard Bernstein-Vazirani circuit with the inner-product oracle leaves the joint state \(\lvert s\rangle\otimes\lvert -\rangle\). Thus one oracle query recovers the hidden string.
