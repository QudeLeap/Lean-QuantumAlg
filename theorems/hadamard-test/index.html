Hadamard test

Theorem statement

For a normalized state \(\psi\) and a unitary gate \(U\), the Hadamard test applied to \(\lvert 0\rangle\otimes\psi\) returns control outcome \(0\) with probability \(\bigl(1+\operatorname{Re}\langle\psi,U\psi\rangle\bigr)/2\).
