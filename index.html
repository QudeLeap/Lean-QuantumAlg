Keywords

  1. Bell state preparation

    The Bell-pair circuit prepares the Bell state exactly: \((\operatorname{CNOT})(H \otimes I)\lvert 00\rangle = \operatorname{bell}\), where \(\operatorname{bell}=2^{-1/2}(\lvert 00\rangle+\lvert 11\rangle)\).

  2. Bernstein-Vazirani algorithm

    For every hidden string \(s\), running the Walsh-Hadamard Bernstein-Vazirani circuit with the inner-product oracle leaves the joint state \(\lvert s\rangle\otimes\lvert -\rangle\). Thus one oracle query recovers the hidden string.

  3. GHZ state preparation

    The three-qubit GHZ circuit prepares the GHZ state exactly from \(\lvert 000\rangle\): applying a Hadamard on the first qubit followed by the two CNOT steps yields \(2^{-1/2}(\lvert 000\rangle+\lvert 111\rangle)\).

  4. Hadamard test

    For a normalized state \(\psi\) and a unitary gate \(U\), the Hadamard test applied to \(\lvert 0\rangle\otimes\psi\) returns control outcome \(0\) with probability \(\bigl(1+\operatorname{Re}\langle\psi,U\psi\rangle\bigr)/2\).

  5. SWAP test

    For normalized states \(\psi\) and \(\phi\), the generalized SWAP test returns control outcome \(1\) with probability \(\bigl(1-\lVert\langle\psi,\phi\rangle\rVert^2\bigr)/2\).

  6. Phase kickback

    For a Boolean oracle \(f\), an input basis state \(\lvert x\rangle\), and the target qubit \(\lvert -\rangle\), the XOR oracle acts as a phase oracle: \(O_f(\lvert x\rangle\otimes\lvert -\rangle)=(-1)^{f(x)}\lvert x\rangle\otimes\lvert -\rangle\).

  7. Deutsch-Jozsa algorithm

    Under the Deutsch-Jozsa promise that the oracle is either constant or balanced, the one-query amplitude test reports "constant" if and only if the oracle is constant.

  8. Simon's problem

    For a Simon oracle satisfying the hidden-mask promise with mask \(s\), if the sampled equations are complete for \(s\) and a candidate \(t\) satisfies all sampled equations, then \(t=s\) and \(s\ne0\).

  9. Quantum Fourier transform

    For every register size \(n\), the quantum Fourier transform matrix \(\operatorname{QFT}_n\) is unitary: \(\operatorname{QFT}_n\in U(2^n)\).

  10. Quantum phase estimation

    If the eigenphase \(\varphi\) is the dyadic rational \(j/2^t\), then applying the inverse quantum Fourier transform to the QPE phase state returns the computational-basis state \(\lvert j\rangle\) exactly: \(\operatorname{QFT}_t^{\dagger}\operatorname{phaseState}(t,\varphi)=\lvert j\rangle\).

  11. Quantum signal processing

    For degree \(d\) and polynomials \((P,Q)\), the QSP pair conditions \(\operatorname{IsQSPPair}(d,P,Q)\) hold if and only if there are phases \((\phi_0,\phi_s)\) of length \(d\) whose QSP word realizes the matrix form \(\operatorname{qspMat}(P,Q,x)\) for every \(x\in[-1,1]\).

  12. Trigonometric QSP

    For length \(L\) and trigonometric polynomial data \((A,B)\), the YZZYZ pair conditions \(\operatorname{IsYZPair}(L,A,B)\) hold if and only if there are real angles \((\phi,\theta_0,\phi_0,p_s)\) with \(|p_s|=L\) whose YZZYZ QSP word realizes \(\operatorname{qspMatYZ}(L,A,B,x)\) for every real \(x\).

  13. Amplitude amplification

    In the two-dimensional subspace spanned by good and bad outcomes, if the reflection product acts as the standard amplitude-amplification rotation, then after \(k\) iterations the state is \(\operatorname{amplitudeAmplificationState}(\theta,k)\). The associated good-outcome probability is \(\sin((2k+1)\theta)^2\).

  14. Grover search

    In the two-dimensional subspace spanned by marked and unmarked states, if diffusion after the phase oracle equals the standard amplification step, then \(k\) Grover iterates send the initial state to \(\operatorname{amplitudeAmplificationState}(\theta,k)\). Measuring a marked state succeeds with probability \(\sin((2k+1)\theta)^2\).

  15. Transformation on controlled unitary

    If \(u\) is an eigenstate of \(U\) with eigenphase \(\theta\), then the YZZYZ quantum phase processor built from controlled \(U\) acts on \(\psi\otimes u\) as the corresponding single-qubit trigonometric QSP matrix at \(\theta\), up to the global phase \((e^{i\theta/2})^L\), tensored with \(u\).

  16. Linear combination of unitaries

    For nonnegative coefficients \(\alpha_i\), a prepare gate \(V\) with first-column amplitudes \(\sqrt{\alpha_i/\lambda}\), and selected unitaries \(U_i\), the projected LCU walk block encodes \(\lambda^{-1}\sum_i \alpha_i U_i\) on the system register.

  17. Quantum teleportation

    For every qubit \(\alpha\lvert0\rangle+\beta\lvert1\rangle\), the teleportation premeasurement branches into four classical outcomes, and for each outcome \((a,b)\) the corresponding correction recovers the original input qubit exactly.

  18. Superdense coding

    For classical bits \(a,b\), encoding them into one half of a shared Bell pair and then decoding yields the computational-basis pair \(\lvert b\rangle\otimes\lvert a\rangle\).

  19. Amplitude estimation

    In the exact dyadic regime, if \(\theta=\pi j/2^t\), amplitude estimation combines QPE readout \(\lvert j\rangle\) with the amplitude-amplification plane identity, giving good-outcome probability \(\sin^2(\pi j/2^t)\).

  20. Eigenvalue phase kickback

    If \(u\) is an eigenstate of \(U\) with eigenvalue \(e^{i\theta}\), then controlled \(U\) maps \((a\lvert0\rangle+b\lvert1\rangle)\otimes u\) to \((a\lvert0\rangle+e^{i\theta}b\lvert1\rangle)\otimes u\).

  21. Order finding

    In the exact case where \(r\mid2^t\), \(\gcd(s,r)=1\), and the QPE outcome index is \(j=s(2^t/r)\), inverse-QFT readout returns \(\lvert j\rangle\) and classical postprocessing recovers the order by \(2^t/\gcd(j,2^t)=r\).

  22. Quantum kernel methods

    For any finite indexed data family \(x_i\) and quantum feature map \(\phi\), the Gram matrix with entries \(K_{ij}=\operatorname{quantumKernel}(\phi,x_i,x_j)\) is positive semidefinite.
